### 安装注意
脚本为了实现简易漫画模式，默认会在所有页面上运行，为此特意将依赖库改为了通过 `@resource` 引入，只在需要用到时才动态加载，尽量减少了在其他页面加载的开销，实测对页面的影响只有 0.XXX 秒。
但如果有心理洁癖，~~在看到了这段后~~对此感到了不舒服，可以自行将脚本头部注释中的 ```// @include   *``` 替换为
```// @include   *```
